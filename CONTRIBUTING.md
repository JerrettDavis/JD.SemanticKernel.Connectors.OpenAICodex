# Contributing

Thank you for your interest in contributing!

## Getting Started

1. Fork and clone the repository
2. Run `dotnet restore` to restore dependencies
3. Run `dotnet build` to verify everything compiles
4. Run `dotnet test` to run all tests

## Pull Requests

- Keep changes focused and minimal
- Ensure all tests pass
- Run `dotnet format` before committing
- Use conventional commit messages

## Code Style

This project uses `.editorconfig` for code style enforcement. Your IDE should
pick up the rules automatically. CI will verify formatting via `dotnet format`.
